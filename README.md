## DM Toolkit 

Written in Next.JS using a fully custom design implemented in tailwindcss. 

## Local Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view the application.
